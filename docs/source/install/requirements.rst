.. _install_requirements:

============
Requirements
============

* Ubuntu 14.04

Required Components
===================

* PHP 5.6, PHP 7
* Nginx
* Cassandra 2.2 or greater
* RabbitMQ
* OpenSSL

Optional Components
===================

* MongoDB *(for boost)*
* Neo4j 2.2 or greater *(for discovery)*
* Redis *(for pubsub, live and caching)*
